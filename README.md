# 📰 Stock News Scraper & AI Analysis Prompt Generator

> **경제·증시 뉴스를 자동으로 수집하고,  
AI가 바로 분석할 수 있는 프롬프트 파일을 생성하는 파이썬 프로젝트**

---

## 📌 프로젝트 소개

이 프로젝트는 네이버 **경제 뉴스 섹션**에서  
의미 있는 뉴스만 자동으로 수집한 뒤,

- 언론사 필터링
- 키워드 필터링
- 기사 본문 요약(첫 문단)
- 발행 시각 수집

을 거쳐  
👉 **AI 분석에 바로 사용할 수 있는 프롬프트 파일**을 생성합니다.

> ❗ AI를 직접 호출하지 않기 때문에  
> **무료로 사용 가능**하며,  
> 나중에 OpenAI API 등으로 확장도 가능합니다.

---

## ✨ 주요 기능

- ✅ 네이버 경제 섹션 뉴스 자동 수집
- ✅ 경제·증시 중심 언론사 필터링
- ✅ 주식 / 금리 / 반도체 / AI 키워드 필터링
- ✅ 기사 본문 첫 문단 요약 추출
- ✅ 발행 시각 + 수집 시각 기록
- ✅ 뉴스 데이터 JSON 파일 저장
- ✅ **AI 분석용 프롬프트 텍스트 자동 생성**

---

## 🛠️ 기술 스택

- **Python 3.x**
- `requests`
- `beautifulsoup4`

---

## 📂 프로젝트 구조

```
stock-news-scraper/
 ├─ main.py                 # 메인 실행 파일
 ├─ analysis_prompt.txt     # AI 분석용 프롬프트 결과물
 ├─ news_YYYY-MM-DD.json    # 수집된 뉴스 데이터
 ├─ requirements.txt
 └─ README.md
```

---

## ⚙️ 실행 방법

### 1️⃣ 라이브러리 설치

```bash
pip install -r requirements.txt
```

### 2️⃣ 실행

```bash
python main.py
```

### 3️⃣ 결과

실행이 완료되면 아래 파일들이 생성됩니다.

- `news_YYYY-MM-DD.json`  
  → 수집된 뉴스 데이터
- `analysis_prompt.txt`  
  → ChatGPT 등 AI에 바로 붙여넣을 수 있는 분석 프롬프트

---

## 🧠 사용 예시

1. 집 PC에서 `main.py` 실행
2. 생성된 `analysis_prompt.txt` 파일 열기
3. ChatGPT에 그대로 붙여넣기
4. **오늘 시장 상황 + 개별 뉴스 분석 결과 확인**

> 회사 PC에 파이썬이나 라이브러리를 설치할 필요 없이  
> **결과 파일만 옮겨서 사용 가능**
